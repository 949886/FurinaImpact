# FurinaImpact
用于某游戏的服务器后端重新实现。

### 构建
可以使用 .NET 7 SDK 进行构建

### 运行
1. 启动 FurinaImpact.SDK 和 FurinaImpact.Gameserver
2. 此时 服务器已正在运行

### 连接
要连接到 FurinaImpact，您可以使用 FurinaImpact.Proxy。只需启动它并设置您的系统代理设置即可。SDK 请求将被重定向。
