# FurinaImpact
あるゲームのためのサーバーバックエンドの再実装。

### ビルド
.NET 7 SDKを使用してビルドできます。

### 実行
1. FurinaImpact.SDKとFurinaImpact.Gameserverを起動します。
2. サーバーが実行されています。

### 接続
FurinaImpactに接続するには、FurinaImpact.Proxyを使用できます。それを起動して、システムのプロキシ設定を行います。SDKリクエストはリダイレクトされます。
